# 🚀 QueueFlow

> Smart QR-Based Queue Management & Ordering System for High-Volume Food Counters

QueueFlow is a full-stack web application that modernizes food ordering and queue management for college canteens, food courts, cafes, juice shops, and restaurants.

Customers scan a QR code to browse the menu, place orders, make online payments, and track order status in real time without waiting in long queues.

---

## ✨ Features

- 📱 QR-based ordering
- ⚡ Real-time order tracking
- 💳 Online payments
- 🔔 Live notifications
- 👨‍💼 Admin dashboard
- 👷 Worker dashboard
- 📊 Analytics dashboard
- 📡 Socket.IO real-time updates
- 🔘 ESP32 IoT support
